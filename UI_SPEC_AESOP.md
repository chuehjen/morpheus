# Morpheus — UI 视觉规范（Aesop 风格）

## 1. 风格目标

Morpheus 不是一个“炫技型 AI 工具”，而是一个安静、可信、像生活方式品牌出品的梦境记录产品。

参考方向：
- Aesop 官网 / 包装的克制感
- 编辑型生活方式品牌的留白与排版
- 高级感来自 **材质、比例、留白、文案克制**
- 不是来自发光、渐变、玻璃质感和夸张动效

---

## 2. 视觉关键词

- 极简
- 安静
- 质感
- 克制
- 温暖
- 自然
- 可信
- 专注内容

---

## 3. 色彩系统

### 主背景
- `#F5F1E8` 页面主背景（暖米白）
- `#EFE8DC` 次级背景 / 区块背景
- `#E5DDCF` 边框浅色

### 文字
- `#1E1A16` 主文字
- `#5A5146` 次级文字
- `#8A7F72` 弱化说明文字

### 强调色
- `#2B2926` 主按钮 / 深色圆形录音按钮
- `#B8AA92` 柔和浅棕标签背景
- `#D8CCBA` 输入框边线 / 分割线
- `#C9BAA4` 成功态浅暖色图形

### 状态色（极弱化）
- 成功：`#7E8B72`
- 错误：`#A06A5F`
- 警告：`#A98B5E`

### 禁止
- 禁止大面积纯黑 `#000`
- 禁止高饱和紫色 / 霓虹蓝 / 荧光绿
- 禁止强烈科技感渐变

---

## 4. 字体建议

优先使用系统字体，保持可实现性：

```css
font-family:
  "Iowan Old Style",
  "Palatino Linotype",
  "Times New Roman",
  serif;
```

正文和 UI 可混用系统无衬线：

```css
font-family:
  -apple-system,
  BlinkMacSystemFont,
  "SF Pro Text",
  "Helvetica Neue",
  Arial,
  sans-serif;
```

### 使用建议
- Logo / 品牌名 / 大标题：衬线风格
- 正文 / 标签 / 按钮：无衬线

---

## 5. 布局规则

- 移动端基准宽度：390px
- 页面水平留白：20px
- 大区块之间留白：24px ~ 32px
- 模块内边距：16px
- 圆角：
  - 卡片：14px
  - 按钮：12px
  - 大圆录音按钮：999px
- 边框：1px solid `#D8CCBA`
- 阴影：极轻，尽量少用

---

## 6. 首页规范

### 页面结构
1. 顶部品牌区：`Morpheus`
2. 主标题：两行内，情绪安静
3. 副标题：一句话说明价值
4. 中央大圆录音按钮
5. AI 价值说明卡片
6. 底部 Tab：记录 / 故事集

### 首页文案建议
- 主标题：`早安，\n记录你的梦境`
- 副标题：`醒来的片刻，让灵感不被溜走。`
- 按钮辅助文案：`点击开始录音`
- 卡片标题：`AI 帮你整理重点`

### 录音按钮
- 深色实心圆
- 中央白色麦克风图标
- 尺寸：120px~136px
- 外圈只能有极轻微柔和描边，不可发光

---

## 7. 录音中页面

- 视觉核心改为 **细线波形**
- 不要夜店式跳动霓虹
- 计时器放中上部
- 停止按钮要干净、克制
- 页面文案短而直接：`正在录音`

---

## 8. AI 解析中页面

- 使用颗粒感圆形加载图案，或微妙点阵扩散
- 文案：
  - 主标题：`AI 正在解析你的梦境`
  - 副标题：`通常需要 10–30 秒`
- 下面列出步骤：
  - 语音转文字
  - 理解梦境内容
  - 提炼主题与元素
  - 生成摘要

---

## 9. 确认记录页

### 结构顺序
1. 标题：`确认你的梦境记录`
2. 心情下拉
3. 主题标签
4. 元素标签
5. 摘要 textarea
6. 原文 textarea
7. 操作按钮：`重新录音` / `保存记录`

### 细节
- 表单不要像后台系统，要像编辑器
- 标签用浅暖灰底
- 文本框高度舒展，不要拥挤
- 主按钮深色，次按钮浅底描边

---

## 10. 保存成功页

- 不使用鲜绿色大勾
- 用浅暖色圆形背景承托深色勾线
- 主按钮：`查看故事集`
- 次按钮：`返回首页`

---

## 11. 故事集页

### 列表样式
- 搜索框置顶
- 每条记录是安静的浅底卡片
- 信息顺序：
  1. 日期
  2. 标签
  3. 摘要首句

### 空状态
- 文案：`还没有梦境故事`
- CTA：`去记录第一则梦`

---

## 12. 设置页

- 显示 Google 绑定状态
- 显示当前 Sheet 名称
- 提供：
  - 绑定 / 重新绑定 Google
  - 重建数据表
  - 关于 Morpheus

风格保持克制，不要出现“开发者工具页”感

---

## 13. 动效原则

- 总体：少、慢、轻
- 页面切换：淡入 / 上移 12px 内
- 录音波形：平缓呼吸感
- 解析 loading：细腻扩散
- 禁止夸张弹跳、飞入、强缩放

---

## 14. 前端实现提示

建议 Claude 把颜色做成 CSS Variables：

```css
:root {
  --bg: #F5F1E8;
  --bg-soft: #EFE8DC;
  --border: #D8CCBA;
  --text: #1E1A16;
  --text-secondary: #5A5146;
  --text-muted: #8A7F72;
  --button-dark: #2B2926;
  --tag-bg: #B8AA92;
  --success: #7E8B72;
  --error: #A06A5F;
}
```

并将页面拆成可复用块：
- `app-header`
- `hero-title`
- `record-button`
- `info-card`
- `bottom-nav`
- `tag-chip`
- `field-group`
- `primary-button`
- `secondary-button`

---

## 15. 禁止事项

- 禁止通篇使用纯黑白高对比
- 禁止科技感紫蓝渐变
- 禁止玻璃拟态
- 禁止厚重阴影
- 禁止把它做成企业后台或 AI 聊天工具
- 禁止“历史”命名，统一使用“故事集”
