# Morpheus 更新包（Aesop UI 版）

这个文件夹是根据最新方向整理出的交付文档，可直接给 Claude 用来写代码。

## 文件说明

- `PRD.md`：更新后的产品需求文档
- `UI_SPEC_AESOP.md`：Aesop 风格 UI 视觉规范
- `CLAUDE_CODE_PROMPT.md`：可直接丢给 Claude 的开发指令

## 建议用法

1. 把这 3 个文件覆盖/补充到你的 GitHub 项目 `morpheus/` 目录
2. 将 `CLAUDE_CODE_PROMPT.md` 内容发给 Claude
3. 让 Claude 先输出静态前端版本
4. 再逐步接入 Web Speech / Worker / Google Sheets API

## 注意

- 命名已统一为「故事集」
- 视觉已切换为 Aesop 风格
- 文档已补充页面、样式、实现约束
